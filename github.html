GitHub es una plataforma de desarrollo colaborativo que permite a los desarrolladores crear, almacenar, gestionar y compartir su código. Utiliza el software de control de versiones Git, proporcionando control de versiones distribuido, control de acceso, seguimiento de errores, solicitudes de características de software, gestión de tareas, integración continua y wikis para cada proyecto1.
